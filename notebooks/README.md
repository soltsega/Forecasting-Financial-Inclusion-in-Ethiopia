# Notebooks
This directory contains Jupyter notebooks used for data analysis, exploration, and modeling.

- `01_data_exploration_and_enrichment.ipynb`: Exploration of the starter dataset and integration of new data.
- `02_exploratory_data_analysis.ipynb`: Detailed EDA and visualization.
- `03_event_impact_modeling.ipynb`: Modeling the impact of various events on financial inclusion indicators.
- `04_forecasting.ipynb`: Time series forecasting for 2025-2027.
